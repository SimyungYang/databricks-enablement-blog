# AI Vibe Coding Workshop — Smart TV 시나리오

> **최종 업데이트**: 2026-04 (예정)

## 개요

**Databricks Intelligence Platform**의 핵심 기능 12가지를 하루 만에 체험하는 핸즈온 워크샵입니다. 스마트TV에서 발생하는 170만 건 이상의 가상 로그 데이터를 기반으로, 데이터 엔지니어링부터 GenAI 에이전트 구축까지 전 과정을 직접 구현합니다.

## 대상

* Databricks 신규 도입 고객 — 플랫폼 전체 기능 체험
* Data Engineer, Data Analyst, ML Engineer
