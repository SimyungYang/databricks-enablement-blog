# Genie Code 사용 가이드

> **최종 업데이트**: 2026-03-27

Databricks Genie Code는 AI 기반 코딩 어시스턴트로, Notebook, SQL Editor, Dashboard, Lakeflow Pipeline 등에서 자연어로 코드를 생성하고 디버깅할 수 있습니다.

## 지원 제품 영역

| 제품 영역 | Chat 모드 | Agent 모드 |
|-----------|-----------|------------|
| **Notebooks** | O | O |
| **SQL Editor** | O | - |
| **Dashboards** | O | O |
| **Lakeflow Pipelines Editor** | O | O |
| **MLflow** | O | - |
