## 개요

Databricks의 생성형 AI 도구인 **Genie Code** 와 **AI Dev Kit** 를 소개하는 자료입니다. 비전문가도 데이터 전문가가 될 수 있는 AI 기반 도구를 다룹니다.

## 대상

* Databricks 신규/기존 고객 — GenAI 도구 도입 검토
* 데이터 팀 리더, BI 분석가

## 핵심 메시지

1. 모든 산업이 AI의 혜택을 받지만, 대부분의 조직은 환경이 파편화
2. Databricks Lakehouse + Unity Catalog로 통합 거버넌스 구축
3. Genie, Genie Code를 통한 AI 기능 제공
4. **생성형 AI로 비전문가도 데이터 전문가가 될 수 있다**
