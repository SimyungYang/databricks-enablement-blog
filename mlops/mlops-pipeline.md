# MLOps on Databricks — End-to-End Pipeline

> **최종 업데이트**: 2026-03-27

## 개요

Databricks에서 MLOps 파이프라인을 구축하는 End-to-End 실습 자료입니다. Feature Engineering부터 Model Monitoring까지 전 과정을 다룹니다.

## 대상

* ML Engineer, Data Scientist
* MLOps 도입을 검토하는 팀

## 노트북 목차

| # | 노트북 | 내용 |
|---|--------|------|
| 01 | Overview | MLOps 개요 및 아키텍처 |
| 02 | Structured Feature Engineering | 정형 데이터 피처 엔지니어링 |
| 03 | Model Training | 모델 학습 (+ 멀티 알고리즘 비교, 고급 기법) |
| 04 | Model Registration (UC) | Unity Catalog에 모델 등록 |
| 05 | Challenger Validation | Challenger 모델 검증 |
| 06 | Batch Inference | 배치 추론 |
| 07 | Unstructured Anomaly Detection | 비정형 데이터 이상 탐지 |
| 08 | Model Monitoring | 모델 모니터링 |
| 09 | MLOps Agent | MLOps 에이전트 |
| 10 | Job Scheduling | 작업 스케줄링 |
